# 扇贝扩展插件

## 下载

[点我下载](https://github.com/jlice/shanbay-extension/raw/master/download/shanbay-extension.crx)

## 功能

### 查词并加入扇贝

任意网站双击取词，查询释义，可以添加到生词，如果该单词已在用户词库中，显示熟悉度并可以选择忘记。
